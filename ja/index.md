# RubyPico

![rubypico-icon](/images/rubypico-icon.png)

小さな端末でも楽しくプログラミングできるRuby開発環境です。ファイルビューワ、エディタ、実行環境を含みます。

- タスクの自動化
- アプリケーションの起動
- チャットボット
- Web API
- 数値計算
- 画像フィルタ

```ruby
class Chat
  def initialize
    @num = 0
  end

  def welcome
    "Hello, world!"
  end

  def call(input)
    @num += 1
    "#{@num}: #{input}"
  end
end
```

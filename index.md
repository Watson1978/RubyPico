# PictRuby

![pictruby-icon](images/pictruby-icon.png)

Ruby Programming Environment in iOS. File Viewer, Editor and Execution Environment included.

- Automate the task
- Launch application
- Web API
- Calculation
- Image filter
- URL Scheme (pictruby://a_script_name)
